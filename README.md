# duolingo-ios-audio-bug-junior-qa
Junior QA bug report case: Duolingo iOS audio issue
# Баг-репорт Duolingo iOS - Junior QA кейс

## О кейсе
Этот репозиторий содержит баг-репорт, оформленный в рамках практики Junior QA.

Описание проблемы:
в мобильном приложении Duolingo на iOS отсутствует звук в упражнениях на аудирование,
при этом интерфейсные звуки работают корректно.


## Что демонстрирует кейс
- Умение описывать баги понятным языком
- Проверку на разных устройствах
- Чёткие шаги воспроизведения
- Фиксацию ожидаемого и фактического результата
- Внимание к пользовательскому опыту


## Кратко о баге
- Платформа: iOS
- Приложение: Duolingo
- Тип бага: функциональный / аудио
- Серьёзность: высокая (блокирует прохождение уроков)
